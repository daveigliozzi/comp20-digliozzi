Security Assessment of Game Center

Introduction

Ming Chow Industries hired me to evaluate the security of the web application Game Center. Game Center was developed to take in data from the new, very popular game 2048, developed by Gabriele Cirulli, and use it to create a High Score leader board. Unfortunately, there are vulnerabilities in Game Center that make it unacceptable as a web application.

Methodology

To find vulnerabilities in Game Center, I first conducted what is referred to as “black box” testing. “Black box” testing involves using tools such as proxies to try to modify code or gain access to the application’s information. I used the proxy Burp Suite for this phase. After “black box” testing, I went into the actual code to look for vulnerabilities myself.
